## 1. 画面仕様一覧

### 画面遷移マップ

```
HomeScreen
  └─→ ShoppingListScreen（買い物リスト）
        └─→ [AIルート計算ダイアログ]
              └─→ SafetyPledgeScreen ★追加
                    └─→ NavigationScreen（ミッション + 歩行ロック）
                          ├─→ [hazardAlertダイアログ]（危険エリア）

  └─→ HomeScreen（さいしょからもういちど）
```

```dart
// lib/services/point_service.dart
class PointService {
  static const _key = 'total_points';
  
  Future<int> load() async {
    final prefs = await SharedPreferences.getInstance();
    return prefs.getInt(_key) ?? 0;
  }
  
  Future<void> add(int points) async {
    final prefs = await SharedPreferences.getInstance();
    final current = prefs.getInt(_key) ?? 0;
    await prefs.setInt(_key, current + points);
  }
}
```

## 5. Vercel デプロイ仕様

### vercel.json（プロジェクトルートに配置）

```json
{
  "buildCommand": "flutter build web --release",
  "outputDirectory": "build/web",
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        { "key": "Cross-Origin-Opener-Policy", "value": "same-origin" },
        { "key": "Cross-Origin-Embedder-Policy", "value": "require-corp" }
      ]
    }
  ]
}
```

> **注意:** COOP/COEP ヘッダーはカメラ・センサーの SharedArrayBuffer 使用時に必要。  
> 方位センサーのみなら不要な場合もあるが、将来のスキャン機能を見越して設定する。

### デプロイ手順

```bash
# 1. Flutter Web ビルド
flutter build web --release

# 2. Vercel CLI でデプロイ（初回）
npx vercel --prod

# 3. 以降は GitHub 連携で自動デプロイ
#    main ブランチへの push → 自動ビルド → 配信
```

## 6. 方位センサー実装仕様（追加実装）

### Web（Flutter Web）での実装

```dart
// lib/services/compass_service.dart
import 'dart:html' as html;
import 'dart:math';

class CompassService {
  Stream<double> get headingStream async* {
    // iOS: 明示的な権限要求が必要
    if (html.DeviceOrientationEvent.hasProperty('requestPermission')) {
      await html.DeviceOrientationEvent.requestPermission();
    }
    
    await for (final event in html.window.onDeviceOrientation) {
      final alpha = event.alpha; // 0〜360（北=0）
      if (alpha != null) yield alpha;
    }
  }
}
```

### UI 反映（AnimatedRotation で針を回転）

```dart
AnimatedRotation(

  duration: const Duration(milliseconds: 200),
  child: const Icon(Icons.navigation, size: 80, color: AppColors.danger),
)
```

## 7. Gemini API 連携仕様（将来実装★）

### クイズ自動生成

```dart
// lib/services/gemini_service.dart
import 'package:google_generative_ai/google_generative_ai.dart';

class GeminiService {
  final _model = GenerativeModel(
    model: 'gemini-2.0-flash',
    apiKey: const String.fromEnvironment('GEMINI_API_KEY'),
  );

  Future<QuizData> generateQuiz(String productName) async {
    final prompt = '''
商品名: $productName
この商品について、小学1〜3年生向けの食育・地産地消クイズを1問作ってください。
JSON形式で返してください:
{"question": "問題文", "choices": ["選択肢1","選択肢2","選択肢3","選択肢4"], "correctIndex": 0, "explanation": "解説"}
''';
    final response = await _model.generateContent([Content.text(prompt)]);
    return QuizData.fromJson(jsonDecode(response.text!));
  }
}
```

> **API キー管理:** `flutter build web --dart-define=GEMINI_API_KEY=xxx` で埋め込む。  
> Vercel の Environment Variables に設定すること。本番では BFF（Vercel Function）経由を推奨。
